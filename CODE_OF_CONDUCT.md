# Code of Conduct

## Our Pledge

We, as contributors, maintainers, and members of the Sansu UI community, pledge to make participation in our project a welcoming, respectful, and harassment-free experience for everyone.

We are committed to fostering an open and inclusive community regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual orientation.

Our goal is to create a collaborative environment where everyone feels comfortable learning, contributing, and sharing ideas.

---

# Our Standards

Examples of behavior that contributes to a positive environment include:

* Being respectful and considerate.
* Using welcoming and inclusive language.
* Providing constructive feedback.
* Accepting constructive criticism gracefully.
* Helping new contributors.
* Being patient with people who are learning.
* Respecting different viewpoints and experiences.
* Collaborating professionally.
* Focusing on what is best for the community.

---

# Unacceptable Behavior

The following behaviors are not acceptable within the Sansu UI community:

* Harassment or discrimination of any kind.
* Personal attacks or insulting comments.
* Hate speech.
* Bullying or intimidation.
* Threats or violent behavior.
* Trolling or deliberately disruptive behavior.
* Sharing another person's private information without permission.
* Spam or malicious content.
* Any conduct that would reasonably be considered inappropriate in a professional environment.

---

# Scope

This Code of Conduct applies to all project spaces, including but not limited to:

* GitHub Issues
* GitHub Discussions
* Pull Requests
* Project Documentation
* Community Chats
* Social Media interactions related to Sansu UI
* Any public representation of the project

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it by opening a private communication with the project maintainer.

# Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, pull requests, issues, discussions, or other contributions that violate this Code of Conduct.

Depending on the severity of the violation, maintainers may take one or more of the following actions:

1. Private warning.
2. Request for correction.
3. Temporary suspension from participation.
4. Permanent ban from the community.

---

# Enforcement Guidelines

Maintainers will make every reasonable effort to evaluate reported incidents fairly and consistently.

The severity of the response will depend on:

* The impact of the behavior.
* Whether the behavior was intentional.
* Whether it is a repeated violation.
* The overall health and safety of the community.
