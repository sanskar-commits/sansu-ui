# Support

Thank you for using **Sansu UI**! 💙

This document explains how to get help, ask questions, report bugs, and contribute to the project.

---

# Before Asking for Help

Before opening an issue or asking a question, please check the following:

* Read the project **README.md**
* Check the component documentation
* Search existing GitHub Issues
* Search existing GitHub Discussions (when available)
* Make sure you're using the latest version of the library

Many common questions have already been answered.

---

# Getting Help

If you need help using Sansu UI, here are the best places to start.

## Documentation

The official documentation website is currently under development.

In the meantime, you can refer to:

* README.md
* Component README files
* Playground examples

---

## GitHub Discussions (Recommended)

For general questions, ideas, or discussions, please use **GitHub Discussions** once it is enabled.

Examples:

* How do I use a component?
* Best practices
* Styling questions
* Design discussions
* Project ideas

GitHub Discussions help keep the issue tracker focused on bugs and feature requests.

---

# Reporting Bugs

If you've found a bug, please open a GitHub Issue.

A good bug report should include:

* Sansu UI version
* React version
* Operating system
* Browser (if applicable)
* Steps to reproduce
* Expected behavior
* Actual behavior
* Screenshots or videos (if applicable)

Providing complete information helps us resolve issues faster.

---

# Feature Requests

Have an idea for a new component or improvement?

We'd love to hear it.

When submitting a feature request, please include:

* The problem you're trying to solve
* Your proposed solution
* Alternative approaches you've considered
* Example code or mockups (optional)

Thoughtful feature requests help us build a better design system.

---

# Security Issues

If you believe you've discovered a security vulnerability, **please do not create a public GitHub Issue**.

Instead, follow the instructions in:

**SECURITY.md**

This allows us to investigate and fix the issue before public disclosure.

# Frequently Asked Questions

## Which React versions are supported?

React 19 and newer.

---

## Is TypeScript supported?

Yes.

Sansu UI is built with TypeScript and provides first-class type definitions.

---

## Is the library production ready?

The project is under active development.

While the published packages are usable, APIs may evolve until the first stable **v1.0.0** release.

---

## Can I contribute new components?

Absolutely!

Please read **CONTRIBUTING.md** before opening a Pull Request.

---

## How do I report documentation issues?

Documentation improvements are always welcome.

Please open a GitHub Issue or submit a Pull Request.
