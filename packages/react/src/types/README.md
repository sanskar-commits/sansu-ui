# Sansu UI Types

This folder contains reusable TypeScript utilities shared across Sansu UI.

## Files

### common.ts

Contains reusable interfaces.

Example:

```ts
interface BaseProps
```

---

### polymorphic.ts

Contains helper types for polymorphic components.

Example:

```tsx
<Box as="section" />

<Button as="a" />
```

---

These types are shared across the entire design system.
