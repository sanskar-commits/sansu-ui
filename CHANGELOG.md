# [0.1.0] - 2026-06-28

🎉 **Initial Public Release**

This is the first public release of **Sansu UI**.

## Added

### Foundation

* Monorepo architecture with PNPM Workspaces
* Turborepo build system
* React package
* Playground application
* Documentation application
* TypeScript support
* TSUP build configuration
* ESLint configuration

### Design System

* Design Tokens

  * Colors
  * Spacing
  * Radius
  * Shadows
  * Typography
  * Breakpoints
  * Sizes
  * Opacity
  * Z-Index
  * Durations
  * Easings
  * Transitions

* Unified Theme Object

### Components

# Versioning Policy

Sansu UI follows **Semantic Versioning**.

## Patch Releases

Example:

```
0.1.1
```

Used for:

* Bug fixes
* Documentation updates
* Small improvements

---

## Minor Releases

Example:

```
0.2.0
```

Used for:

* New components
* New features
* Backward-compatible improvements

---

## Major Releases

Example:

```
1.0.0
```

Used for:

* Breaking changes
* Stable API milestones
* Significant architectural improvements
